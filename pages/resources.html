Key Takeaways

  1. AI is a compressed knowledge base - great for common info, but verify important facts
  2. Tools extend AI's capabilities - web search for current info, code for calculations
  3. Multiple modalities - text, voice, and images all work
  4. Start simple - try ChatGPT first for writing and planning
  5. Use Perplexity for research - when you need current info with sources
  6. Iterate and refine - your first prompt doesn't have to be perfect
  7. AI assists, you decide - your judgment and local knowledge are irreplaceable
